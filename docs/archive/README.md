# Archive: Early Development Planning Documents

This directory serves as an archive reference for planning documents created during the early development phase of Platinum Casino (October 2025). These documents remain in the repository root for historical reference but have been largely superseded by the structured documentation in `docs/`.

## Purpose

During the initial development sprint in October 2025, several planning and review documents were created at the repository root to guide development priorities, track bugs, and plan the migration from MongoDB to MySQL. As the project matured, a comprehensive documentation system was established under `docs/` with organized sections covering architecture, features, API reference, and more.

The root-level planning documents are preserved as-is because:

1. **Historical context** -- They capture the original design decisions and priorities at the time they were made.
2. **Migration record** -- They document the MongoDB-to-MySQL migration process and the ESM conversion effort.
3. **Issue tracking** -- Some items in these documents may still be relevant and are tracked in the structured docs under `docs/11-roadmap/`.
